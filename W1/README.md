# Week 1 - C# Fundamentals

## Giới thiệu

Đây là project tổng hợp các bài tập trong tuần đầu tiên của quá trình học C# và .NET 8.

Mục tiêu của project:

- Làm quen với cú pháp C#
- Thực hành Console Application
- OOP cơ bản
- Thiết kế class
- Thao tác với List<T>
- Xây dựng ứng dụng CRUD đơn giản
